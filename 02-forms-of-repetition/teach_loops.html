The Double For Loop

One loop walks across columns. Another walks down rows. Together they visit every cell in a grid.

Step 1

The idea in one sentence

Imagine moving your hand across a sheet of grid paper — left to right across a row, then dropping down to start the next row. That's the double for loop. The outer loop picks a row r. The inner loop walks every column c in that row, left to right. Together, they visit every single cell.

Think of r as which row you're on, and c as how far across that row you are.

      Step 3
      
How pixel coordinates are calculated

    

    
The grid divides the canvas into equal cells. c and r are just index numbers (0, 1, 2…). To get actual pixel positions, multiply by the cell size:


    

      

        
coordinate math

        
let col_width  = width / num_cols;   // 400 / 5 = 80
let row_height = height / num_rows;  // 400 / 5 = 80

// Inside the loop:
let x = c * col_width;   // left edge of cell
let y = r * row_height;  // top edge of cell

// Center of the cell:
x + col_width  / 2
y + row_height / 2


      

      

        
So when c = 2 and r = 1:

        
x = 2 × 80 = 160

        
y = 1 × 80 = 80

        
Ellipse center: (200, 120)

        

          The cell index × cell size gives you the top-left corner. Add half the cell size to get the center.

sketch.js — full version

        
let PALETTE = ["#FFFFFF", "#000000"];

function setup() {
  createCanvas(400, 400);
  noLoop();
}

function draw() {
  let num_rows = 5;
  let num_cols = 5;

  let col_width  = width  / num_cols;
  let row_height = height / num_rows;

  background(220);

  // Outer loop — walks down rows
  for (let r = 0; r < num_rows; r++) {

      // Inner loop — walks left to right across columns
      for (let c = 0; c < num_cols; c++) {

        let x = c * col_width;
        let y = r * row_height;

        fill(random(PALETTE));
        ellipse(x+col_width/2,
               y+row_height/2,
               30, 30);
    }
  }
}
