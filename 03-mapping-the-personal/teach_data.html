From Sleep Data to Art

Turn your weekly sleep log into generative visuals using objects and map().

Step 1

Data Collection: Your Weekly Sleep

For the last 7 nights, you wrote down these 3 data points for each night:

# Data Point Type What you wrote
1 Hours Slept Number e.g. 8, 7.5, 6
2 Restfulness Number (1–3) 1 = Exhausted   2 = Okay   3 = Refreshed
3 Dreamed? Boolean true = Yes    false = No
Step 2

The Goal: Visualizing Your Data

We want to create a sketch where the visual properties of the circles are controlled by your real sleep data:

Data PointControlsHow it looks
Hours Slept Number of circles More hours = more rings
Restfulness Stroke Weight (thickness) Thicker lines = more rested
Dreamed? Circle Color true = bright blue   false = grey

But here is the problem: How do we store all of these different pieces of data together in our code? How do we tell the computer that this specific number of hours, this restfulness, and this dream status all belong to Monday night?

Step 3

The Solution: Objects

In JavaScript, this structure is officially called an Object.

Think of it like a real dictionary — each key (the word) has a value (the definition). You look up a key to get its value.

Fun Fact: Depending on the programming language, you might hear this structure called a Dictionary (in Python), a Hash Map, or an Associative Array. They all mean basically the same thing!
Key = The label or name (like hours)
Value = The data stored under that label (like 8)

Instead of creating separate variables that have no connection to each other, we can group them all into one object:

// Clean and organized in an object! let sleepData = { "day": "Mon", "hours": 8, // Controls the number of circles "restful": 3, // Controls the stroke weight "dreamed": true // Controls the color };
Syntax Rules:
• Wrap the whole thing in curly braces { }
• Each entry is a key : value pair
• Separate entries with commas
• To read a value later, use dot notation: sleepData.hours

Step 5

Expanding to 7 Days: A List of Objects

So far we have represented one night of sleep as a single object. But we collected data for 7 nights. How do we store all of them?

The answer is a list (called an Array in JavaScript). We can put multiple objects inside an array using square brackets [ ], with each object separated by a comma:

let sleepLog = [ { "day": "Mon", "hours": 8, "restful": 3, "dreamed": true }, { "day": "Tue", "hours": 6, "restful": 1, "dreamed": false }, { "day": "Wed", "hours": 7, "restful": 2, "dreamed": true }, { "day": "Thu", "hours": 5, "restful": 1, "dreamed": false }, { "day": "Fri", "hours": 7, "restful": 2, "dreamed": true }, { "day": "Sat", "hours": 9, "restful": 3, "dreamed": false }, { "day": "Sun", "hours": 8, "restful": 3, "dreamed": true } ];
Key idea: Each item in the array is a complete object with all four keys. We can loop through this array to draw one set of circles for each night!

To access a specific night, use the index (starting from 0):

let monday = sleepLog[0]; // First item → Mon let mondayHours = sleepLog[0].hours; // → 8 let friday = sleepLog[4]; // Fifth item → Fri let fridayDream = sleepLog[4].dreamed; // → true
