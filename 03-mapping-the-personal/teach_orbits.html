Adding Social Orbits

Visualize your daily social interactions as an orbit diagram.

Step 1

Data Collection: Your Social Orbits

For each of the last 7 days, write down the people you interacted with in person and score each person on this scale:

1 = Family (closest)
2 = Friend
3 = Acquaintance
4 = Stranger (farthest)

Here is an example of what your table might look like for one day:

DayPersonCloseness Score
MonMom1 (Family)
MonBest friend2 (Friend)
MonClassmate3 (Acquaintance)
MonBarista4 (Stranger)
Note: You don't need to remember names — just the closeness score for each person. A day where you saw your mom, a friend, and a stranger would be: [1, 2, 4]
Step 2

Adding the "people" Key to Your Data

We already have our sleep data stored as objects. Now we just need to add one new key called people to each day. The value is an array of closeness scores:

// Before — just sleep data { "day": "Mon", "hours": 8, "restful": 3, "dreamed": true } // After — with social data added! { "day": "Mon", "hours": 8, "restful": 3, "dreamed": true, "people": [1, 2, 3, 4] }
What "people" stores: An array of numbers. Each number is one person you saw that day.
The length of the array = how many people you interacted with.
Each value (1–4) = how close that person is to you.
Step 3

How the Orbit Drawing Works

We want to draw you as a dot in the center, and each person you interacted with as a small dot orbiting around you. Closer relationships orbit nearer to the center:

ConceptWhat it doesCode
Angle Spread people evenly around the circle map(k, 0, numPeople, 0, TWO_PI)
Distance 1 (Family) = closest to you, 4 (Stranger) = farthest away map(closeness, 1, 4, 45, 65)
Position Convert angle + distance into x, y coordinates cos(angle) * distance, sin(angle) * distance
Think of it like a solar system: You are the "sun" in the center. Family members orbit close (like inner planets), and strangers orbit farther out (like distant planets).

Here is the code that draws the orbit diagram. This goes inside the outer loop, right after the sleep circle code:

// ── SOCIAL ORBITS ────────────────────────────────────── strokeWeight(1.5); // Draw "Self" dot in the center of this day's column fill(50); circle(center_X, center_Y, 6); noFill(); // How many people did we interact with today? let numPeople = dayData.people.length; // Loop through each person for (let k = 0; k < numPeople; k++) { // Get this person's closeness score (1–4) let closeness = dayData.people[k]; // Spread people evenly in a full circle (0 to TWO_PI) // e.g. with 4 people: 0°, 90°, 180°, 270° let angle = map(k, 0, numPeople, 0, TWO_PI); // Family (1) orbits close, Strangers (4) orbit far // 1 → 45px from center, 4 → 65px from center let distance = map(closeness, 1, 4, 45, 65); // Convert the angle + distance into x, y position // cos() gives the horizontal offset // sin() gives the vertical offset let person_X = center_X + cos(angle) * distance; let person_Y = center_Y + sin(angle) * distance; // Faint connecting line back to "Self" stroke(200); line(center_X, center_Y, person_X, person_Y); // Draw the person as a small filled dot stroke(50); fill(240); // Fill to cover the line behind the dot circle(person_X, person_Y, 6); noFill(); }
