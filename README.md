# Form / Flow / Feel

Taught at [School for Poetic Computation (SFPC)](https://sfpc.study/), Spring 2026.

---

This course is about making invisible systems visible. Over five weeks, students move from building a shared visual vocabulary to developing a self-defined visual system grounded in personal, environmental, or ecological data.

class arc:

```
01  Systems & Sensibilities
     ╲
      ╲_____________________
                             02  Forms of Repetition
                                          ╲
                                           ╲
                                            ╲
                                  03  Mapping the Personal
                                              │
                                              │
                                              │
                                      04  World as Input
                                            ╱
                                           ╱
                           05  Assemblies & Reflections
```

In-class demos use [p5.js](https://editor.p5js.org/). Students are welcome to work in any tool that fits their practice.
